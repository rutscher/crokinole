A: Done + Undo Side by Side

Bottom-right has both Undo and Done. After tapping Done, it becomes a green "✓ Waiting..." indicator. Undo stays available so you can still fix mistakes while waiting. Board stays interactive until both confirm.

B: Done Replaces Undo

Bigger Done button in the corner (no separate Undo). After tapping, it becomes "✓ Waiting..." with a tiny "tap to undo" hint — tapping the waiting indicator un-confirms and goes back to Done. Cleaner, one button.

How it works

  • Each player taps Done when they've finished recording their discs
  • Button changes to ✓ Waiting... — shows they've confirmed
  • When both players are confirmed, round auto-ends and score calculates
  • Board stays interactive (undo/remove discs) even while waiting — tapping waiting un-confirms
  • End Round moves to the menu for the rare case of ending early by agreement

Recommendation: B — one button that toggles between Done and Waiting is simpler. The disc-level undo (tap-to-remove) is still available on the board itself, and tapping the waiting state lets you un-confirm if you need to change something.
